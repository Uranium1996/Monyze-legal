1. 한 줄 요약

모나이즈는 어떠한 개인정보도 수집하지 않습니다. 사용자가 입력한 가계부 데이터는 전부 사용자 기기 내부에만 저장됩니다. 회사 또는 제3자에게 전송되는 데이터는 없습니다.

2. 수집하는 개인정보

회사는 다음의 개인정보를 수집하지 않습니다.

  • 이름, 이메일, 전화번호, 주소 등 식별 정보
  • 광고 식별자(IDFA / GAID)
  • 위치 정보, 연락처, 사진 등 기기 데이터
  • 사용 기록, 분석 / 트래킹 데이터

3. 앱 내부에 저장되는 데이터 (기기 로컬)

아래 정보는 앱 내부 SQLite 데이터베이스에 저장되며 사용자 기기를 벗어나지 않습니다.

  • 거래 내역(날짜, 금액, 카테고리, 메모, 계좌)
  • 사용자가 정의한 카테고리 / 계좌 / 환율 설정
  • 언어 / 통화 등 환경 설정 값
  • PIN 비밀번호: SHA-256 해시 처리 후 OS 보안 저장소(iOS 키체인 / Android Keystore)에 저장

4. 네트워크 통신

앱은 일반 사용 중에는 인터넷을 사용하지 않습니다. 네트워크를 사용하는 경우는 다음의 단 한 가지입니다.

  • 오류 신고 / 의견 보내기 기능을 사용할 때, 사용자가 작성한 본문이 기기의 메일 앱을 거쳐 회사 이메일 (Uraniumura1996@gmail.com) 로 발송됩니다. 이 과정은 사용자가 직접 송신 버튼을 누를 때만 발생합니다.

5. 데이터 보관 및 파기

모든 데이터는 사용자가 앱을 삭제하거나 앱 내 "초기화" 기능을 사용할 때까지 기기에 보관됩니다. 회사는 사용자 데이터를 별도로 보관하지 않으므로 파기 절차도 존재하지 않습니다.

6. 제3자 제공 / 위탁

회사는 사용자 데이터를 제3자에게 제공하거나 처리를 위탁하지 않습니다. 앱은 광고 SDK, 분석 SDK, 푸시 SDK 등을 일절 포함하지 않습니다.

7. 이용자의 권리

  • 모든 데이터는 사용자 기기에 있으므로 언제든 직접 열람·수정·삭제할 수 있습니다.
  • 데이터를 외부로 백업하려면 앱의 엑셀 추출 기능을 사용하면 됩니다.
  • 전체 삭제는 앱을 삭제하거나 OS 의 앱 데이터 삭제 기능을 사용하면 됩니다.

8. 보안

PIN 잠금을 설정하면 앱 진입 시 비밀번호를 확인합니다. PIN 은 평문이 아닌 SHA-256 해시로 저장됩니다. 사용자 기기 자체의 잠금 / 백업 보안은 OS 설정을 따릅니다.

9. 개정

본 방침이 개정되는 경우 앱 내 설정 화면에 변경 일자와 함께 게시합니다. 중요한 변경 사항은 앱 업데이트 노트에서도 안내합니다.

Uranium Studio (the “Company”) treats your privacy with care. This policy describes what information Monyze (the “App”) handles and how that information is stored and protected.

1. TL;DR

Monyze does not collect any personal information. All ledger data you enter is stored entirely on your device. Nothing is transmitted to the Company or any third party.

2. Personal information we do not collect

  • Identifiers: name, email, phone number, address
  • Ad identifiers (IDFA / GAID)
  • Location, contacts, photos, or other device data
  • Usage analytics or tracking telemetry

3. Data stored locally on your device

The following data is stored only inside the App’s local SQLite database and never leaves your device:

  • Transactions you enter (date, amount, category, note, account)
  • Categories, accounts, exchange rates you configure
  • Preferences such as language and base currency
  • PIN passcode: stored as an SHA-256 hash inside the OS secure store (iOS Keychain / Android Keystore)

4. Network usage

The App does not use the internet during normal use. The only case in which it touches the network is:

  • Report a bug / Send feedback: when you press send, the message you typed is passed to your device’s mail app and emailed to Uraniumura1996@gmail.com. This only happens when you explicitly tap send.

5. Data retention & deletion

All data remains on your device until you uninstall the App or use the in-app “Reset” option. Since the Company holds no user data, there is no separate deletion procedure to follow.

6. Third parties

The Company does not share, sell, or outsource user data to any third party. The App contains no advertising SDK, no analytics SDK, and no push-notification SDK.

7. Your rights

  • All data lives on your device — view, edit, or delete it at any time.
  • Use the Excel export feature to back data up externally.
  • Uninstall the App or clear app data via the OS to remove everything.

8. Security

When the PIN lock is enabled, the App requires the passcode on launch. The PIN is never stored in plain text — only its SHA-256 hash is kept in secure storage. Device-level lock and backup security follow your OS settings.

9. Changes to this policy

We will post any revisions to this policy inside the App’s settings with an effective date. Material changes will also be summarized in the App’s release notes.
